## Overview

Autonomous cloud incident detection and remediation agent — AWS CloudWatch, EventBridge, n8n, LangChain, MCP. This project is part of the MAIA — Multi-Cloud AI Architect portfolio, providing reference architectures and reusable building blocks for cloud architects working across AWS, Azure, and GCP.
